### Para rodar o código

#### Cloning
```ps
  # Clone o repositório utilizando o git
  $ git clone (https://github.com/Brenovieira41/Cron.git)
```

#### :arrow_forward: Executando
```ps
  # Para baixar os pacotes
  $ npm install
  # Para executar o código
  $ npm run start:dev
```
